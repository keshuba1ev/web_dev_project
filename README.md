# Web Development Project - Dorama Catalog

This project is a Web-Based Drama/Dorama Catalog containing an Angular frontend and a Django backend.

## Project Description
A comprehensive platform to browse, discover, and track your favorite Asian dramas. The application consists of:
* **Frontend**: A modern, responsive user interface built using Angular.
* **Backend**: A robust REST API powered by Python and Django REST Framework.

## Getting Started

### Backend
1. Navigate to the `backend` directory.
2. Set up your virtual environment and install dependencies.
3. Run migrations and start the server: `python manage.py runserver`

### Frontend
1. Navigate to the `frontend` directory.
2. Run `npm install` to install dependencies.
3. Start the Angular CLI development server: `ng serve`
